Terraform IaC

Application déployée automatiquement

Du code au conteneur en quelques secondes grâce à l’automatisation.

Plan

Terraform simule la création des ressources Docker et affiche les changements prévus.

Apply

Les images sont construites ou récupérées, puis les conteneurs PostgreSQL et Webapp sont lancés.

State

Le fichier d’état garde la trace des ressources afin de garantir l’idempotence du déploiement.
